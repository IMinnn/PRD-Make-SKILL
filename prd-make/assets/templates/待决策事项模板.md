# 待决策事项

## 使用说明

- 本文件用于集中记录 PRD 写作和评审过程中尚未确定、需要产品/技术/业务继续确认的事项。
- 待决策事项不写入 `PRD文档.md` 正文，也不写入功能模块详情。
- 已关闭事项保留在本文件中，通过状态字段标记，不从记录中删除。
- 只有会影响产品行为、工程范围、数据模型、权限、流程、优先级或验收标准的问题才记录为待决策事项。
- 当前处理/默认假设只能作为推进写作的临时处理方式，不能视为用户已确认结论。
- 用户确认后，应更新状态和关闭时间；必要时同步修订 `PRD文档.md`、`模块规划.md` 或 `追加日志.md`。
- ID 使用 `D001`、`D002` 连续编号，不与功能 ID、业务规则 ID 或验收标准 ID 混用。
